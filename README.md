# Daikibo-Machine-Downtime-Dashboard-Tableau-
machine-downtime-analysis-analysis

 Overview
This project focuses on analyzing machine downtime across multiple factories of Daikibo. The goal is to identify areas with high downtime, understand patterns per factory and machine type, and provide insights to improve operational efficiency.
The analysis helps management monitor equipment health, reduce downtime, and optimize production workflows.
Dashboard Insights

The Tableau dashboard provides two key visualizations:

Downtime per Factory
Displays total machine downtime for each factory.
Highlights factories with the highest downtime to prioritize maintenance.

Downtime per Machine
Shows downtime by device type across factories.
